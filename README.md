## Comandos de uso de la app

```
Opciones:
      --help             Muestra ayuda                                [booleano]
      --version          Muestra número de versión                    [booleano]
      --tb, --tablaBase  Es la base de la tabla de multiplicar
                                               [número] [requerido] [defecto: 5]
  -h, --hasta            La cantidad de veces a multiplicar
                                               [número] [requerido] [defecto: 5]
  -l, --listar           Muestra la tabla en consola
                                         [booleano] [requerido] [defecto: false]
```
